# Issue: Integrar CEUCRUD com API publica do IBGE

## Descricao

Adicionar ao CEUCRUD uma consulta de municipios por UF consumindo a API publica de Localidades do IBGE.

## Valor para a aplicacao

A funcionalidade ajuda a equipe academica a consultar municipios oficiais por estado, apoiando planejamento de polos, cadastros e expansao institucional.

## Criterios de aceite

- A aplicacao deve permitir informar uma UF.
- A aplicacao deve buscar municipios na API do IBGE.
- A interface deve exibir os municipios retornados.
- Deve existir teste automatizado validando o fluxo de dados da API.
- O README deve informar o link do deploy e como executar o projeto.
